# Merge-2048
2048 Theme Merging Game built using Python and Pygame

## 🎮 Features

- 🔢 5x5 grid with auto-merging logic
- 🎨 Built with Pygame for graphical interface
- ⌨️ Drag and Drop controls
- 🧠 Auto Merging Mode (optional for auto play)
